# Work Breakdown Structure (WBS)

1. Project Management
    - Kickoff meeting
    - Weekly check-ins
2. Requirements Gathering
    - Interview client
    - Collect existing content
3. Design
    - Create wireframes
    - Approve design with client
4. Development
    - Build site
    - Content migration
    - SEO setup
5. Testing & QA
    - Internal testing
    - Client review
6. Launch
    - Deploy site
    - Handover & training
